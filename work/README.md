# paydayloan

> 贷上钱前端项目

## 项目信息
* tapd : [https://www.tapd.cn/20051441/prong/iterations](https://www.tapd.cn/20051441/prong/iterations)
* [git 仓库](https://git.fond.io/kfq/paydayloanvue)

## 前端项目配置

```
$ // 克隆源代码库
$ git clone git@git.fond.io:kfq/paydayloanvue.git

$ // 安装依赖
$ cd paydayloanvue/
$ npm install
```
